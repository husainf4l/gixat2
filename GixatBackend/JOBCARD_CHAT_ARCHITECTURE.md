```
...
101. SELECT * FROM Users WHERE Id = 'user-100'  -- 100th comment author

```

### âœ… With DataLoaders (Batching)

```
Query: Get 100 comments with authors

1. SELECT * FROM JobCardComments WHERE JobCardId = '123'  -- 100 comments
2. SELECT * FROM Users WHERE Id IN ('user-1', 'user-2', ..., 'user-100')  -- ALL authors

Total: 2 database queries! âœ¨ FAST
```

**DataLoader Benefits:**
- Automatic batching of duplicate requests
- Caching within single request
- Per-request lifecycle (no stale data)
- Works across the entire GraphQL query tree
